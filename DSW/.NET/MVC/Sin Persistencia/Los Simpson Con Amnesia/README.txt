Para acceder a la página web:
1. Tener instalado el Visual Studio 2022 (VS para abreviar)
2. Descargarse el repositorio de la siguiente manera:
    2.1 Con el VS abierto, seleccionar "Clonar Repositorio"
    2.2 En el primer campo, poner el link del repositorio (en Github, el botón verde que
        dice "<> Code" -> apartado https -> copiar link)
    2.3 Darle al botón de Clonar.
3. Teniendo el VS y el repositorio, darle al botón de "Play" (triángulo verde) o presionar f5
    para iniciar la página web.
